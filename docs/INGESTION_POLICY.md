# Ingestion Policy

## Allowed long-term memory content

- durable project state
- branch and baseline summaries
- doctrine updates
- planned work with scope
- normalized findings that remain useful beyond a single chat

## Avoid by default

- full raw chats as canon
- transient noise
- duplicate cells without context fingerprints
- implementation claims not grounded in source state

## Required fields for durable cells

- content
- compact context summary
- scope
- tags
- provenance
- baseline snapshot
