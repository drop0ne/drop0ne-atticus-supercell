# Mirror Policy

## Principle

This repository mirrors selected state from source repos into a normalized memory surface.

## Current mirror targets

- `drop0ne/amas-spec`
- `drop0ne/atticus-mpo`

## Rules

1. Source repos remain authoritative for their own code and primary docs.
2. Mirrors in this repo should summarize, timestamp, and reference source canon.
3. Mirror entries should not silently diverge from source state.
4. If a mirror intentionally interprets or normalizes source content, that should be stated clearly.
