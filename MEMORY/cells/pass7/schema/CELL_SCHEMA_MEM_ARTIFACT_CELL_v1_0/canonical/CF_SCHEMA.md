# CELL_SCHEMA_MEM_ARTIFACT_CELL_v1_0

## Purpose
Memory artifact cell schema

## Canonical rules
- Artifact cells must preserve source lineage, authority class, and derivation status.
- Artifact schemas should make the difference between canonical source and derived artifact explicit.
- Artifact-cell structure exists to support portability and auditability, not to grant canonical status.

## Notes
- Works with AIP and manifest schema for structured artifact generation.
- Important for exports, handoffs, and cross-runtime reuse.
