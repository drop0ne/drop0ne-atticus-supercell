# CELL_SCHEMA_MEM_ARTIFACT_INSTANTIATION_v1_0

## Purpose
Artifact instantiation schema

## Canonical rules
- Instantiated artifacts must declare source cells, derivation path, and authority inheritance.
- Instantiation metadata should disclose whether content is copied, transformed, summarized, or bundled.
- Instantiation structure should support audit and replay without conflating artifact output with source canon.

## Notes
- Complements the artifact-cell schema by governing the act of producing derivative artifacts.
- Important for deterministic handoff and export workflows.
