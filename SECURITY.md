# Security Policy

## Scope

This repository is a public-facing ATTICUS / AMAS supercell reference surface.
It is intended to expose governance, structure, schemas, example artifacts, and validation scaffolding.

It is **not** the right place to disclose:

- private credentials
- secrets or tokens
- private infrastructure details
- unpublished sensitive operator data
- exploit details that create immediate operational risk
- sensitive findings that should not be made public first

## Reporting

Do **not** open a public GitHub issue for sensitive disclosures.

Instead, contact the maintainer privately through the operator-controlled channel you already use.
If no private channel is available, do not publish sensitive exploit details here.

## Expected disclosure posture

For non-sensitive problems, normal public issues or pull requests may be acceptable if public issue intake is enabled.

For sensitive problems:

1. provide a minimal private summary
2. include affected files or surfaces if known
3. include reproduction conditions if safe to share privately
4. avoid publishing secrets, tokens, or exploit chains publicly

## Current maturity note

This repository is an active-development prototype / reference implementation.
Validation and automation surfaces are still evolving.
Treat all findings with appropriate caution until independently verified.
