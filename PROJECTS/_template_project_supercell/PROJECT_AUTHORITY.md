## Authority boundaries

- Source code repo remains authoritative for project code and project-local primary docs.
- Project supercell remains authoritative for project long-term memory, normalized state, and resume artifacts.
- Global ATTICUS supercell remains authoritative for umbrella cross-project continuity.

## Write discipline

- No silent rewrite of durable project memory.
- Add or supersede with traceability.
- Preserve lineage between major milestone cells.

## Required invariants

1. One project-local authority root.
2. Compact context-bearing memory cells only.
3. Mirrors summarize source canon and link back to it.
4. Resume packets must be reproducible from current state and durable memory.
